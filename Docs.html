Cosmetics
GET http://fortnite-api.de/api/cosmetics
Returns a JSON of all cosmetics with information
GET http://fortnite-api.de/api/cosmeticids
Returns a JSON of all cosmetic IDS
GET http://fortnite-api.de/api/cosmetics/search
Returns a JSON of specific cosmetics with information

Paramaters:
?displayname= (name of cosmetic)
?description= (description of cosmetic)
?id= (ID of cosmetic)
?pakfile= (pakfile of cosmetic)
GET http://fortnite-api.de/api/cosmeticimage
Returns an image with information of a cosmetic

Paramaters:
?cosmeticid= (ID of cosmetic)
&featured= (true or false)
GET http://fortnite-api.de/api/cosmeticimage/description
Returns an image with information of a cosmetic with a description

Paramaters:
?cosmeticid= (ID of cosmetic)
&featured= (true or false)
Extraction
GET http://fortnite-api.de/api/extract/json
Returns a JSON with information of a path

Paramaters:
?path= (file path)
GET http://fortnite-api.de/api/extract/texture
Returns an image with the texture of a path

Paramaters:
?path= (file path)
GET http://fortnite-api.de/extract/audio
Returns an ogg with the sound of a path

Paramaters:
?path= (file path)
Information
GET https://fortnite-api.de/api/brshop
Returns a parsed JSON with information of the current BR shop
GET https://fortnite-api.de/api/brstorefronts
Returns a JSON with information of the current BR shop directly from Epic
GET https://fortnite-api.de/api/starterkits
Returns a JSON with information of the current starterkits
GET http://fortnite-api.de/api/eventflags
Returns a JSON with information of the current eventflags
GET http://fortnite-api.de/api/keychain
Returns a JSON with information of the current keychain
AES Keys
GET https://fortnite-api.de/api/aeskeys
Returns a JSON of the current AES keys along with the IDS associated with it
GET https://fortnite-api.de/api/aeskeys/main
Returns a JSON of the current main AES key
